==Description==

Rescue Shortcodes is a lightweight WordPress shortcodes plugin that when activated, will add a button to the WordPress text editor to easily insert shortcodes to your posts or pages.

**Installation**

1. Upload the plugin to `/wp-content/plugins/`. Alternately, you can download this package as a zip and upload it via the WordPress dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click the Rescue Themes icon in the WordPress text editor to display your shortcodes options.

**Available Shortcodes**

* Spacing
* Highlight text
* Buttons
* Alert Box
* Grid Columns
* Toggle Box
* Tab Group
* Google Map
* Font Awesome Icons
* Animate Content
